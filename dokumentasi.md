# 📘 Dokumentasi Portfolio Kelompok 1

## 🛠️ Cara Mengembangkan Proyek

1. Buka `css/style.css` untuk mengubah tampilan.
2. Buka `js/data.js` untuk mengubah data anggota.
3. Buka `js/main.js` untuk memodifikasi logika website.
4. Tambahkan halaman HTML baru jika diperlukan.

---

## 💡 Prompt untuk Membangun Website Portofolio

"Buatkan website portofolio kelompok sekolah dengan tampilan modern dan responsif. Gunakan halaman utama berisi judul, deskripsi singkat, kartu anggota dengan foto, nama, peran, dan tautan sosial. Tambahkan animasi halus untuk kartu dan latar belakang, dan pastikan desain cocok untuk tampilan desktop dan mobile."

### Contoh prompt tambahan

- Buat desain landing page portofolio sekolah dengan gradient lembut
- Buat struktur proyek website portofolio sederhana
- Tambahkan kartu anggota dan media sosial

---

## ✨ Catatan Penting

- Pastikan file HTML, CSS, JS, dan gambar berada di lokasi yang benar.
- Periksa nama file gambar jika tidak muncul.
- Gunakan browser terbaru untuk hasil terbaik.
