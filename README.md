# 🌐 Portfolio TKJ — SMK Rajasa Surabaya

Website portofolio kelompok siswa TKJ dengan tema **Aurora Gradient**.

---

## 📁 Struktur Folder

```
portfolio-tkj/
├── index.html              ← Halaman utama (buka ini di browser)
├── css/
│   └── style.css           ← Semua styling & animasi aurora
├── js/
│   ├── data.js             ← Data anggota kelompok
│   └── main.js             ← Logic render kartu & animasi
├── assets/
│   └── images/
│       ├── nuhshon.jpg
│       ├── bilal.jpg
│       ├── vienel.jpg
│       └── ridho.jpg
└── README.md
```

---

## 🚀 Cara Pakai

1. Download & ekstrak folder `portfolio-tkj`
2. Buka file **`index.html`** di browser (Chrome / Edge / Firefox)
3. Selesai! ✅

> ⚠️ Semua file harus dalam **satu folder** agar foto tampil.
