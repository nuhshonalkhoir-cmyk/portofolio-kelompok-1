Tema website tersebut adalah website portofolio kelompok bertema teknologi modern/futuristik yang dibuat oleh siswa TKJ (Teknik Komputer dan Jaringan).
Desainnya menggunakan nuansa dark mode, warna biru–ungu neon, efek glow, serta tampilan card modern yang memberi kesan profesional seperti website startup atau tim developer.

Tema Website

Website ini mengusung konsep:

Portofolio Tim / Company Profile
→ menampilkan identitas anggota kelompok dan kemampuan masing-masing.
Teknologi & Digital Future
→ terlihat dari penggunaan warna neon, gradient biru, dan kata-kata seperti:
“membangun masa depan digital bersama”.
Modern UI/UX
→ memakai layout card, efek blur/glassmorphism, dan typography tebal yang modern.
Isi Website

Pada halaman yang terlihat, isi website menampilkan:

1. Header / Identitas Tim

Bagian atas berisi:

Nama sekolah:
SMK Rajasa Surabaya – TKJ
Nama tim:
Tim Kernel ID
Deskripsi singkat:
“Empat siswa TKJ yang membangun masa depan digital bersama”

Fungsinya untuk memperkenalkan kelompok dan tujuan website.
