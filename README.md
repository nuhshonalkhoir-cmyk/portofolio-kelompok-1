# 🌐 Portfolio Kelompok 1 — Website Portofolio TKJ

README ini berisi panduan lengkap untuk menjalankan website portofolio kelompok, daftar kontributor, dan contoh prompt untuk membangun halaman portofolio.

---

## 📁 Struktur Proyek

```
portfolio-kelompok-1/
├── index.html              ← Halaman utama portofolio
├── css/
│   └── style.css           ← Styling dan layout website
├── js/
│   ├── data.js             ← Data anggota dan konten dinamis
│   └── main.js             ← Logika rendering dan interaksi
├── assets/
│   └── images/             ← Foto anggota dan aset grafis
│       ├── nuhshon.jpg
│       ├── bilal.jpg
│       ├── vienel.jpg
│       └── ridho.jpg
└── README.md               ← Dokumentasi proyek
```

---

## 🚀 Cara Menjalankan Website

1. Pastikan semua file berada di satu folder utama.
2. Buka file **`index.html`** dari folder proyek.
3. Website akan terbuka di browser secara langsung.

> Jika menggunakan browser, cukup klik dua kali pada `index.html` atau buka melalui menu "Open File".

---

## 🛠️ Cara Mengembangkan Proyek

1. Buka `css/style.css` untuk mengubah tampilan, warna, dan efek.
2. Buka `js/data.js` untuk menambahkan atau memperbarui data anggota.
3. Buka `js/main.js` untuk memodifikasi logika render kartu, animasi, dan interaksi.
4. Jika ingin menambahkan halaman baru, buat file HTML baru dan tautkan ke `index.html`.

## 💡 Prompt untuk Membangun Website Portofolio

Gunakan prompt berikut jika ingin membuat atau memperbarui website portofolio kelompok menggunakan AI atau referensi desain:

"Buatkan website portofolio kelompok sekolah dengan tampilan modern dan responsif. Gunakan halaman utama berisi judul, deskripsi singkat, kartu anggota dengan foto, nama, peran, dan tautan sosial. Tambahkan animasi halus untuk kartu dan latar belakang, dan pastikan desain cocok untuk tampilan desktop dan mobile."

### Contoh prompt tambahan untuk AI desain

- "Buat desain landing page portofolio sekolah dengan gradient lembut, ikon sosial media, dan layout kartu anggota." 
- "Buat struktur proyek website portofolio sederhana dengan file `index.html`, `css/style.css`, `js/main.js`, dan `js/data.js`."

---

## ✨ Catatan Penting

- Pastikan semua file `.html`, `.css`, `.js`, dan folder `assets/images` tetap berada dalam satu folder.
- Jika gambar tidak muncul, cek kembali nama file dan ekstensi dalam folder `assets/images`.
- Gunakan browser terbaru agar animasi dan layout tampil optimal.
